Dice for Crypto Casino

About

Dice is an add-on game for the Crypto Casino application. This game can not be installed on its own without having Crypto Casino installed first.

System requirements

  • Crypto Casino 1.9.0 or higher.

Installation

Step 1

Download the add-on game package from CodeCanyon, unzip it and upload the contents to the web root folder of your domain, where the main Crypto Casino app is installed.

Step 2

Log in to the Crypto Casino backend, open Maintenance page, click "Run database updates" and then "Clear cache".

Step 3

Open Settings page in the Crypto Casino backend and configure Dice game settings (min bet, max bet, default bet etc).

After that the game will become automatically available for play at http://yourwebsite/games/dice

Frequently Asked Questions

How to upgrade the game to a new version?

To update the game to a new version you need to follow the same steps as during installation:

  • Download a new version from CodeCanyon, unzip and copy all files to your server.
  • Log in to the backend, open Maintenance page and click "Run database updates".
  • Click "Clear cache".

The game settings will be preserved.
